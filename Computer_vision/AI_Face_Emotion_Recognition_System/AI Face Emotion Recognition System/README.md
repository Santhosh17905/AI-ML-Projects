# 📌 Overview

The AI Face Emotion Recognition System is an advanced Computer Vision project that identifies human emotions from facial expressions in real time through a webcam feed.

The application leverages facial detection, emotion classification, confidence scoring, analytics tracking, and automated logging to create a professional-grade emotion recognition platform.

This project demonstrates practical applications of:

* Artificial Intelligence
* Computer Vision
* Deep Learning
* Human-Computer Interaction
* Real-Time Video Processing
* Emotion Analytics

# 🏗 System Architecture

```text
Webcam Feed
     │
     ▼
Frame Acquisition
     │
     ▼
Face Detection
(OpenCV Haar Cascade)
     │
     ▼
Face Region Extraction
     │
     ▼
Emotion Classification
     │
     ▼
Analytics Engine
     │
     ├── Emotion History
     ├── CSV Logging
     ├── Statistics
     └── Smart Capture

     ▼

Final Output
```

# ⚙ Installation

## Clone Repository

```bash
git clone https://github.com/santhosh17905

cd AI-Face-Emotion-Recognition
```

---

## Create Virtual Environment

### Windows

```bash
python -m venv venv

venv\Scripts\activate
```

### Linux / Mac

```bash
python3 -m venv venv

source venv/bin/activate
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

# ▶ Running the Project

```bash
python main.py
```

```text
Q
```

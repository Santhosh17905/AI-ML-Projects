# 🚀 Real-Time Object Recognition Using MobileNet SSD


## 📌 Project Overview

This project demonstrates a real-time Object Recognition System using OpenCV's DNN module and the MobileNet SSD pre-trained deep learning model.

The application captures live video from a webcam, detects objects in real-time, draws bounding boxes around detected objects, displays confidence scores, calculates FPS, and provides a professional dashboard interface.

This project is part of my **30 Days of AI & Machine Learning Challenge**.

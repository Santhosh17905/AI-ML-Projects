## ⚙️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/Santhosh17905/AI-ML-Projects.git
```

2. Navigate to project folder:

```bash
cd AI-ML-Projects
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Run any project:

```bash
python filename.py
```

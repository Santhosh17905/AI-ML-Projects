# 🤖 AI-Powered Hand Gesture Recognition System

A real-time Computer Vision project that detects and recognizes hand gestures using OpenCV and MediaPipe. The system tracks hand landmarks, identifies gestures, displays live statistics, and supports screenshot capture and gesture logging.

## 🚀 Features

* Real-Time Hand Tracking
* 21 Hand Landmark Detection
* Gesture Recognition
* FPS Monitoring
* Screenshot Capture
* Gesture Statistics
* Gesture Stabilization
* CSV Logging
* Voice Feedback Support
* Modern UI Overlay

## 🛠️ Technologies Used

* Python
* OpenCV
* MediaPipe
* NumPy
* Pandas
* Pyttsx3

## 🎯 Supported Gestures

| ---------------- | --------- |
| ✊ Fist           | FIST      |

| 🤟 Three Fingers | THREE     |
| 🖖 Four Fingers  | FOUR      |

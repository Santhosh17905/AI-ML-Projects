# ❤️ Heart Disease Prediction Using Naive Bayes

## 📌 Project Overview

This project is a **Machine Learning-based Heart Disease Prediction System** built using the **Naive Bayes Algorithm**.
It predicts whether a person is at risk of heart disease based on medical input parameters.

The project also includes a **modern interactive web application** built using Streamlit with:

* 🎨 Animated GIF background
* 🧊 Glassmorphism UI
* 📊 Risk percentage visualization
* 📈 Progress bar (Heart Risk Meter)

---

## 🚀 Features

* ✅ Data preprocessing & visualization
* ✅ Naive Bayes classification model
* ✅ Accuracy, Precision, Recall, F1-score
* ✅ Risk percentage prediction
* ✅ Interactive Streamlit UI
* ✅ Styled inputs & glowing buttons
* ✅ Real-time prediction system

---

## 🧠 Machine Learning Algorithm Used

* Gaussian Naive Bayes

---

## 📊 Dataset Features

The dataset includes the following attributes:

* Age
* Sex
* Chest Pain Type
* Resting Blood Pressure
* Cholesterol
* Fasting Blood Sugar
* Resting ECG
* Max Heart Rate
* Exercise Induced Angina
* Oldpeak
* ST Slope
* Target (Heart Disease)

## 💡 How It Works

1. User enters health parameters
2. Data is preprocessed using StandardScaler
3. Model predicts:

   * Heart Disease (Yes/No)
   * Risk Percentage
4. UI displays:

   * Risk Score
   * Progress Bar
   * Final Result
